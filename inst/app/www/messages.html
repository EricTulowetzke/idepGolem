Major improvements in iDEP 1.0:

  • Completely rewritten using Shiny modules with the Golem framework,
  • Multiple demo datasets ready to be loaded,
  • Generates R code to reproduce DEG analysis,
  • Generates html reports for some analyses.
  • Interactive heatmaps with enrichment analysis,
  • Download high-quality figures throughout the app.

iDEP is still under active development and testing. We welcome any suggestions or questions. Please Email Jenny or report issues on our GitHub page.
